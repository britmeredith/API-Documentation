Content Sentiment Analyzer API (2.0.0)

Download OpenAPI specification:

Meredith: meredith@example.com License: Apache 2.0

A powerful API for analyzing the sentiment and toxicity of text content. This API supports single and batch analysis, with asynchronous processing for large workloads. It uses OAuth2 for authentication.

Analyze a single text comment

Analyzes the sentiment and toxicity of a single piece of text.

Authorizations:
oauth2
Request Body schema: application/json
required
text
required
string

The text to analyze.

Responses

Production Server

https://api.example.com/v2/analyze

Sandbox Server for testing

https://sandbox-api.example.com/v2/analyze

Analyze a batch of text comments

Submits a batch of up to 50 text comments for asynchronous analysis. Returns a jobId that can be used to poll for the results.

Authorizations:
oauth2
Request Body schema: application/json
required
texts
required
Array of strings [ 1 .. 50 ] items

An array of strings to analyze.

Responses

Production Server

https://api.example.com/v2/analyze/batch

Sandbox Server for testing

https://sandbox-api.example.com/v2/analyze/batch

Request samples

  • Payload
Content type
application/json
{
  • "texts": [
    • "This is a great comment.",
    • "This is a hateful and toxic comment.",
    • "I don't know what to say."
    ]
}

Get the status of a batch analysis job

Poll this endpoint to get the status and results of a batch analysis job.

Authorizations:
oauth2
path Parameters
jobId
required
string <uuid>

The ID of the batch analysis job.

Responses

Production Server

https://api.example.com/v2/status/{jobId}
